# Data Analysis-Chocolate Shipment Data
This project analyzes chocolate shipment data to uncover sales trends, shipment performance, and customer demand. Using Excel, the data was cleaned, transformed, and visualized to create an interactive dashboard showcasing top regions, high-demand products, shipment volume, and revenue insights for better business decisions.

## Objective of the Project
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
The main objective of this project is to:
Analyze chocolate sales performance across different countries
Evaluate product-wise and sales person-wise performance
Track shipment and box distribution efficiency
Identify high-performing markets and products
Create interactive dashboards for better business decision-making
